Clorurarea metanului

Halogenarea alcanilor are loc printr-un mecanism homolitic înlanțuit: este o reactie de substitutie, care are loc prin intermediul radicalilor, cunoscuta sub numele de substituție radicalică.
În reacția dintre metan si clor in raport molar 1 : 1, se formeaza preponderent monoclorometanul:
CH4(substrat) + Cl2(reactant) -> CH3Cl + HCl

a
Mecanismul homolitic înlănțuit parcurge trei etape

Initierea - are loc cu scindarea homolitică a moleculei de reactant ( Cl2)in conditii fotochimice:
Cl2 -> 🟠 Cl(homoliza)
🟠 Cl -> Cl2(coligre)
Atomii liberi de clor formați au reactivitate mult mai mare decât clorul in stare moleculară(Cl2).

Propagarea - este etapa in care atomii de clor rezultați în sistem reacționează cu moleculele de metan formând radicali liberi metil (CH3) care, având reactivitate mare, la ciocnirea cu o moleculă de clor, formează molecule de CH3Cl si un atom liber de clor (🟠Cl); atomul de clor reia lanțul de reacție, procesul se repetă continuu, atât timp cât apar particule de natură radicală (🟠CH3 și 🟠Cl):
CH4 + 🟠Cl -> 🟠CH3 + HCl
🟠 CH3 + Cl2 -> CH3Cl + 🟠 Cl
CH4 +🟠 Cl -> 🟠CH3 + HCl
🟠CH3 + Cl2 -> CH3Cl +🟠 Cl
........................................
Întreruperea - reprezintă etapa in care particulele cu caracter radicalic din sistem se unesc între ele(coligare) formând molecule stabile, iar lanțul de reacții nu mai continuă:
🟠CH3 +🟠 Cl -> CH3 - Cl
🟠 CH3 + 🟠CH3 -> CH3 - CH3
🟠Cl +🟠 Cl -> Cl2
.......................................
Obținerea celorlalți compuși clorurați ai metanului se formează tot prin mecanism homolitic înlanțuit (SR), asemănător celui descris anterior.

b
Utilizari ale compusilor clorurați ai metanului

-clorura de metil se folosește ca agent frigorific și agent de metilare.

-clorura de metilen, cloroformul si tetraclorura de carbon sunt utilizati ca solvenți pentru uleiuri și grăsimi.

-clorura de metilen este folosită și ca decapant pentru piese metalice.

-tetraclorura de carbon este folosita ca agent pentru stingerea incendiilor.
